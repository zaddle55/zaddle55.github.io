2.4 Equivalence Relation 等价关系

When RRR is a binary relation on set AAA and RRR is a equivalence relation, RRR has following properties:

  • R is Reflexive(自反) meaning for all x∈Ax \in Ax∈A, xRxxRxxRx.
  • R is Symmetric(对称) meaning for all x,y∈Ax, y \in Ax,y∈A, if xRyxRyxRy then yRxyRxyRx.

e.g. “=” is an equivalence relation on N\mathbb{N}N.

2.6 Partial Order Relation 偏序关系

A binary relation RRR is a partial order when satisfying RRR is:

  • Reflective
  • Antisymmetric
  • Transitive

Minimal element: for x∈Ax \in Ax∈A there doesn’t exist another y∈Ay \in Ay∈A such that yRxyRxyRx, then xxx is the minimal element.

Maximal element: for x∈Ax \in Ax∈A there doesn’t exist another y∈Ay \in Ay∈A such that xRyxRyxRy, then xxx is the maximal element.

If RRR is a partial order relation and for any x,y∈Ax, y \in Ax,y∈A there exists xRyxRyxRy or yRxyRxyRx, then RRR is a total order relation over AAA

3. Function 函数

A function from XXX to YYY is a binary relation RRR that satisfies:

  • For all x∈Xx \in Xx∈X, there exists y∈Yy \in Yy∈Y such that xRyxRyxRy.
  • If y,zy,zy,z \in Y such that xRyxRyxRy and xRzxRzxRz, then y=zy=zy=z.

3.1 Notation 记法

We typically use f,g,hf,g,hf,g,h to represent functions

f:A→Bf: A \to B f:A→B

expresses that fff is a function from set AAA to BBB.

  • Domain(定义域): the set of input values (AAA for fff).
  • Codomain(陪域): the set of possible output values(BBB for fff).
  • Range(值域): the set of actual output values (subset of BBB for fff).

f:A1×A2×⋯×An→Bf: A_1 \times A_2 \times \dots \times A_n \to Bf:A1​×A2​×⋯×An​→B is called an n-ary function

An n-ary function maps ordered n-tuples from its domain to elements in its codomain.
