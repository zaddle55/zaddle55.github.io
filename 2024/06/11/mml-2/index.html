1. Syntax 语法

1.1 Proposition 命题

A declarative sentence that can be judged as either true or false is called Proposition.

1.1.1 Atomic Proposition 原子命题

The proposition can’t be devided into a smaller one.

Typically symbolized as p,q,r,…p, q, r, \dotsp,q,r,…

1.1.2 Compound Proposition 复合命题

The proposition is built by two or more atomic proposition is called composition proposition.

Typically symbolized as A,B,C,…A, B, C, \dotsA,B,C,…

1.2 Logical Connectives 逻辑连接词

Connectives Meaning Term
∧\land∧ and conjuction
∨\lor∨ or disjunction
¬\neg¬ not negation
→\to→ if…then… implication
↔\leftrightarrow↔ if and only if equivalence

1.3 Alphabet 符号系统

  • Atomic proposition: p,q,r,…p, q, r, \dotsp,q,r,…
  • Logic connectives: ∧,∨,¬,→,↔\land,\lor,\neg,\to,\leftrightarrow∧,∨,¬,→,↔
  • Punctuations: ( , )

1.4 Expressions 表达式

Finite strings of symbols is called expression.

Empty expression is denoted by λ\lambdaλ.

1.5 Formulas 公式

Definition for Atom(LP)Atom(\mathscr{L}^P)Atom(LP): A set of expressions consisting of one symbol only.

Induction definition for Form(LP)Form(\mathscr{L}^P)Form(LP):

  1. Atom(LP)⊆Form(LP)Atom(\mathscr{L}^P) \subseteq Form(\mathscr{L}^P)Atom(LP)⊆Form(LP).

  3. If A,B∈Form(LP)A, B \in Form(\mathscr{L}^P)A,B∈Form(LP), then (A∧B),(A∨B),(A→B),(A↔B)∈Form(LP)(A \land B), (A \lor B), (A \to B), (A \leftrightarrow B) \in Form(\mathscr{L}^P)(A∧B),(A∨B),(A→B),(A↔B)∈Form(LP).

A well-formed formula(WFF) can be and only be derived by this way.

1.6 Precendence 优先顺序

  • Connectives: ¬,∧,∨,→,↔\neg,\land,\lor,\to,\leftrightarrow¬,∧,∨,→,↔ (Less priority from left to right).
  • Parentheses (with punctuations"()") take the highest precendence.
  • With connectives of the same precedence, the rightmost occurrence have more priority (e.g. p→q→r≡p→(q→r)p \to q \to r \equiv p \to (q \to r)p→q→r≡p→(q→r) ).

Let A∈Form(LP)A \in Form(\mathscr{L}^P)A∈Form(LP)

2.4 Logic Equivalence 逻辑等价

Formulas A,BA, BA,B are logically equivalent iff ∀v,Av=Bv\forall v, A^v=B^v∀v,Av=Bv.

2.4.1 Laws for Logic Equivalence 逻辑等价律

  • Identity laws:
    • p∧T≡pp \land T \equiv pp∧T≡p
    • p∨F≡pp \lor F \equiv pp∨F≡p
  • Domination laws:
    • p∨T≡Tp \lor T \equiv Tp∨T≡T
    • p∧F≡Fp \land F \equiv Fp∧F≡F
  • Idempotent laws:
    • p∨p≡pp \lor p \equiv pp∨p≡p
    • p∧p≡pp \land p \equiv pp∧p≡p
  • Double negation laws:
    • ¬(¬p)≡p\neg(\neg p) \equiv p¬(¬p)≡p
  • Commutative laws:
    • p∨q≡q∨pp \lor q \equiv q \lor pp∨q≡q∨p
    • p∧q≡q∧pp \land q \equiv q \land pp∧q≡q∧p
  • Associative laws:
    • (p∨q)∨r≡p∨(q∨r)(p \lor q) \lor r \equiv p \lor (q \lor r)(p∨q)∨r≡p∨(q∨r)
    • (p∧q)∧r≡p∧(q∧r)(p \land q) \land r \equiv p \land (q \land r)(p∧q)∧r≡p∧(q∧r)
  • Distributive laws:
    • p∨(q∧r)≡(p∨q)∧(p∨r)p \lor (q \land r) \equiv (p \lor q) \land (p \lor r)p∨(q∧r)≡(p∨q)∧(p∨r)
    • p∧(q∨r)≡(p∧q)∨(p∧r)p \land (q \lor r) \equiv (p \land q) \lor (p \land r)p∧(q∨r)≡(p∧q)∨(p∧r)
  • De Morgan’s law:
    • ¬(p∨q)≡¬p∧¬q\neg(p \lor q) \equiv \neg p \land \neg q¬(p∨q)≡¬p∧¬q
    • ¬(p∧q)≡¬p∨¬q\neg(p \land q) \equiv \neg p \lor \neg q¬(p∧q)≡¬p∨¬q
  • Absorption laws:
    • p∨(p∧q)≡pp \lor (p \land q) \equiv pp∨(p∧q)≡p
    • p∧(p∨q)≡pp \land (p \lor q) \equiv pp∧(p∨q)≡p
  • Negation laws:
    • p∨¬p≡Tp \lor \neg p \equiv Tp∨¬p≡T
    • p∧¬p≡Fp \land \neg p \equiv Fp∧¬p≡F
  • Implicative laws:
    • p→q≡¬p∨qp \to q \equiv \neg p \lor qp→q≡¬p∨q
  • Contrapositive laws:
    • p→q≡¬q→¬pp \to q \equiv \neg q \to \neg pp→q≡¬q→¬p
  • Equivalence laws:
    • p↔q≡(p→q)∧(q→p)p \leftrightarrow q \equiv (p \to q) \land (q \to p)p↔q≡(p→q)∧(q→p)

2.4.2 Substitution 代换

BBB is a substitution instance of AAA iff B may be obtained from AAA by substituting formulas for propositional variables in AAA, replacing each occurrence of the same variable by an occurrence of the same formula.

e.g. (p→p)↔(p→p)(p \to p) \leftrightarrow (p \to p)(p→p)↔(p→p) is a substitution instance of q↔qq \leftrightarrow qq↔q

Theorem 111: A,B∈(LP)A, B \in (\mathscr{L}^P)A,B∈(LP). If AAA is a tautology and BBB is a susbtitution instance of AAA, then BBB is again a tautology.

Theorem 222: A∈Form(LP)A \in Form(\mathscr{L}^P)A∈Form(LP). AAA contain a subformula CCC (i.e., CCC is a segment of AAA and is itself a well-formed formula). If C≡DC \equiv DC≡D, then replacing some occurrences (not necessarily all) of the subformula CCC in AAA with DDD to obtain the formula BBB, then A≡BA \equiv BA≡B.

3. Entailment 蕴涵

Let Σ⊆Form(LP),A∈Form(LP)\Sigma \subseteq Form(\mathscr{L}^P), A \in Form(\mathscr{L}^P)Σ⊆Form(LP),A∈Form(LP). We say:

  • AAA is a logical consequence of Σ\SigmaΣ
  • Σ\SigmaΣ entails AAA
  • Σ⊨A\Sigma \models AΣ⊨A

Iff for all truth valuation vvv if Σv=1\Sigma^v=1Σv=1 then Av=1A^v=1Av=1

Also, Σ⊭A\Sigma \not\models AΣ⊨A denotes there exists a truth valuation vvv such that Σv=1\Sigma^v=1Σv=1 and Av=0A^v=0Av=0.

3.1 Prove Entailment 蕴涵证明

  • Direct Proof: For every truth valuation under which all of the premises are true, show that the conclusion is also true under this valuation.
  • Using Truth Table: Consider all rows of the truth table in which all of the formulas in Σ\SigmaΣ are true. Verify that AAA is true in all of these rows.
  • Proof by Contradiction Assume that the entailment does not hold, which means that there is a truth valuation under which all of the premises are true and the conclusion is false. Derive a contradiction.
